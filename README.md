# MarianaJudithRA.github.io
Este proyecto es una página web que simula una tienda en línea, creada como parte de mis prácticas de programación. La tienda incluye varias funciones que permiten al usuario explorar productos, añadir articulos al carrito o favoritos, y acceder mediante un sistema básico de login.

Funciones principales:
Catálago de productos con varias categorías.
Carrito de compras.
Sección de favoritos.
Sistema de login básico.

Tecnologías utilizadas:
HTML5: Estructura de la tienda.
CSS: Estilos, colores, y diseño.
JavaScript: Funcionalidad del carrito, favoritos y login.

Este proyecto me permitió reforzar mis conocimientos en HTML, CSS y JavaScript, además de entender como funcionan las interacciones en una tienda en línea, Gracias!
